## 🧠 CV / NLP 大模型基础论文

### 1. HALP：无需生成即可检测 VLM 幻觉

- 提出一种无需模型生成任何 token 即可检测视觉-语言模型幻觉的方法
- 发表于 EACL 2026，直接在模型内部表征上操作
- **要点**：避免了传统「先生成再验证」的高成本流程，实现实时幻觉检测
- **影响**：为 VLM 部署中的安全性与可靠性提供了轻量级解决方案
- 🔗 [arXiv:2603.05465](https://arxiv.org/abs/2603.05465)

### 2. Towards Multimodal Lifelong Understanding：多模态终身理解数据集与 Agentic 基线

- 构建面向多模态终身理解的数据集，并提出 Agentic 基线方法
- 融合视觉、语言和时序信息的长程理解任务
- **影响**：推动多模态 AI 从「单次推理」走向「持续理解」
- 🔗 [arXiv:2603.05484](https://arxiv.org/abs/2603.05484)

### 3. Wiki-R1：基于数据与采样课程的多模态推理激励（ICLR 2026）

- 针对知识密集型 VQA 提出数据课程与采样课程相结合的训练策略
- 被 ICLR 2026 接收，代码和权重已公开
- **影响**：为知识型视觉问答提供了高效的推理增强路径
- 🔗 [arXiv:2603.05256](https://arxiv.org/abs/2603.05256)

### 4. Mario：基于多模态图推理的 LLM（CVPR 2026）

- 将图结构推理引入多模态大模型，增强结构化关系理解
- 被 CVPR 2026 接收
- **影响**：弥合了 LLM 在结构化视觉推理上的短板
- 🔗 [arXiv:2603.05181](https://arxiv.org/abs/2603.05181)

### 5. Planning in 8 Tokens：紧凑离散 Tokenizer 用于潜在世界模型（CVPR 2026）

- 仅用 8 个 token 表示规划信息，实现极度紧凑的潜在世界模型
- **要点**：大幅降低规划模块的计算开销，保持决策质量
- **影响**：为机器人/自动驾驶等场景的高效规划提供新范式
- 🔗 [arXiv:2603.05438](https://arxiv.org/abs/2603.05438)

## 🤖 Agent 相关论文

### 1. WideSeek：多 Agent 扩展实现广度研究（Wide Research）

- 提出从 Deep Research 到 Wide Research 的新范式
- 构建 WideSeekBench 基准和动态层级多 Agent 架构，可自主 fork 并行子 Agent
- **要点**：解决复杂约束下的广度信息检索与综合
- **影响**：拓展了 Agent 的搜索与研究能力边界
- 🔗 [arXiv:2602.02636](https://arxiv.org/html/2602.02636v1)

### 2. MA-CoNav：主从式多 Agent 协作导航框架

- 受分布式认知理论启发，提出「主从」层级 Agent 协作架构
- 解耦并分配长距离视觉-语言导航中的认知负荷
- **影响**：为复杂长程具身导航提供了可扩展的多 Agent 解决方案
- 🔗 [arXiv:2603.03024](https://arxiv.org/pdf/2603.03024)

### 3. LLandMark：地标感知的多 Agent 多模态视频检索

- 模块化多 Agent 框架，包含查询解析、地标推理、多模态检索和排序合成四阶段
- 地标知识 Agent 将文化/空间地标重构为视觉描述提示
- **影响**：展示了领域知识 Agent 在视频理解中的价值
- 🔗 [arXiv:2603.02888](https://arxiv.org/html/2603.02888v1)

### 4. TraderBench：对抗性资本市场中 AI Agent 的鲁棒性评测

- 提交至 ICLR 2026 Agents in the Wild Workshop
- 评估 AI Agent 在对抗性金融环境中的表现
- **影响**：填补了 Agent 在高对抗性真实场景中的评测空白
- 🔗 [arXiv:2603.00285](https://arxiv.org/abs/2603.00285)

### 5. Multi-Sourced Multi-Agent Evidence Retrieval for Fact-Checking

- 多源多 Agent 证据检索用于事实核查
- 多个专业 Agent 协作从不同来源获取证据
- **影响**：为 AI 辅助内容审核和虚假信息检测提供了架构参考
- 🔗 [arXiv:2603.00267](https://arxiv.org/abs/2603.00267)
