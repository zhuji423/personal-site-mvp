### 1. Anthropic 年化营收突破 300 亿美元，正式超越 OpenAI 登顶 AI 收入王座

- **核心做了什么**：Anthropic 宣布其年化运营收入（run-rate revenue）已突破 300 亿美元，较 2025 年底的 90 亿美元增长超 3 倍，较 2 月底的 190 亿美元在一个月内新增 110 亿美元。这是 AI 行业有史以来最快的营收增长速度——同等规模下年化增长率约 9,700%。这一数字正式超越 OpenAI 此前报告的 250 亿美元 ARR。
- **启示 / To-dos**：
    - Claude Code 和企业订阅是核心驱动力，"一旦签了年度合同，切换到 Codex 并不容易"——先发锁定效应已经形成
    - AI 行业收入从"概念验证"到"指数爆发"的拐点已至，所有基建投资假设需要重新校准
    - 关注 Anthropic IPO 时间线——多位分析师认为此次公告为 IPO 铺路
    - 对比 Salesforce 花了 20+ 年才达到类似收入规模，Anthropic 不到 3 年
- **正面**：验证了 AI 产品的真实商业需求（而非泡沫）；编码 Agent 成为新的"杀手应用"品类；Anthropic 从"安全研究机构"到"收入机器"的转型成功
- **负面 / 风险**：run-rate 是瞬时指标而非累计收入，短窗口内波动可能很大；单月 110 亿增量的可持续性存疑；"$65 成本服务 $20 订阅"的经济模型是否可持续；巨额收入若主要来自编码工具，集中度风险高

### 2. Anthropic 签署 Google/Broadcom 数 GW 级 TPU 协议，AI 算力格局重塑

- **核心做了什么**：Anthropic 与 Google 和 Broadcom 签署了一项多 GW（gigawatt）级别的下一代 TPU 算力协议，预计 2027 年起上线。Broadcom 同日向 SEC 提交 8-K 文件，确认与 Google 达成长期协议生产未来版本 TPU，并扩大对 Anthropic 的算力供应至约 3.5 GW。
- **启示 / To-dos**：
    - TPU 作为 NVIDIA GPU 的替代方案正在获得战略性采纳——"dedicated silicon 的崛起"正在发生

    - Broadcom 的定制芯片战略获得里程碑式验证
    - 对 NVIDIA 股价和市场预期可能产生中期压力
- **正面**：算力供应多元化降低行业对单一供应商的依赖；Google Cloud 通过 TPU 获得差异化竞争优势
- **负面 / 风险**：2027 年才上线意味着当前算力缺口仍将持续；多 GW 级数据中心的能源和冷却需求是巨大挑战

### 12. Karpathy 提出"LLM 知识库"范式：知识编译取代即时检索

- **核心做了什么**：Karpathy 在 X 上发布长帖，详细介绍他使用 LLM 构建个人知识库的方法论：将原始文档（论文、文章、代码、数据集）索引到 raw/ 目录，用 LLM 渐进式"编译"为结构化 wiki（.md 文件），包含摘要、反向链接、概念分类和文章。他的一个研究主题 wiki 已达约 100 篇文章、40 万字。使用 Obsidian 作为前端 IDE。
- **启示 / To-dos**：
    - "知识编译"vs "即时检索（RAG）"是两种不同范式——前者投入更大但查询质量更高
    - 在小规模（~100 篇文章）下，LLM 自维护的索引文件 + 摘要已经可以替代 fancy RAG
    - 这一方法论可直接应用于团队知识管理和技术文档维护
- **正面**：极简但有效；不需要复杂的 RAG 基础设施；知识复用价值随时间指数增长
- **负面 / 风险**：编译过程中的信息损失和偏差会随时间累积；40 万字规模可能是当前方法的上限
- **原文链接**：[Karpathy on X: LLM Knowledge Bases](https://x.com/karpathy/status/2039805659525644595)
- **对研发/工程启示（Karpathy 视角）**：核心洞察："LLM 写和维护所有 wiki 数据，我几乎不直接触碰它。" 这是"人做判断，AI 做编译"哲学的又一实践。知识管理的未来不是搜索，而是持续编译。
